# Quick Start Guide

Get EnvFly CLI up and running in just 5 minutes! This guide will walk you through the essential steps to start managing your environment variables securely.

<Steps>
  <Step title="Install EnvFly CLI">
    Choose your preferred installation method:
    <Tabs>
      <Tab title="npm">```bash npm install -g envfly-cli ```</Tab>
      <Tab title="yarn">```bash yarn global add envfly-cli ```</Tab>
      <Tab title="Homebrew">```bash brew install envfly-cli ```</Tab>
      <Tab title="Direct Download">
        ```bash curl -fsSL
        https://github.com/adarsh-technocrat/EnvFly-CLI/releases/latest/download/envfly-cli.sh
        | bash ```
      </Tab>
    </Tabs>
  </Step>

  <Step title="Authenticate">
    Login to EnvFly Cloud Service: ```bash envfly login ``` You'll be prompted
    for your API key. Get one from [app.envfly.io](https://app.envfly.io).
  </Step>

  <Step title="Initialize Project">
    Navigate to your project and initialize EnvFly: ```bash cd your-project
    envfly init ``` This creates a `.envfly` configuration file.
  </Step>

  <Step title="Add Environment Variables">
    Create your first environment: ```bash # Add variables to development
    environment envfly push development
    DATABASE_URL="postgresql://localhost:5432/dev" envfly push development
    API_KEY="dev_key_123" # Add variables to production environment envfly push
    production DATABASE_URL="postgresql://prod-server:5432/prod" envfly push
    production API_KEY="prod_key_456" ```
  </Step>

  <Step title="Pull Variables">
    Pull environment variables to your local `.env` file: ```bash # Pull
    development environment envfly pull development # Or pull production (be
    careful!) envfly pull production ```
  </Step>
</Steps>

## Troubleshooting

<Callout type="info">
  If you encounter any issues during setup, check our [troubleshooting
  guide](/troubleshooting/common-issues) or join our [Discord
  community](https://discord.gg/envfly) for help.
</Callout>

<AccordionGroup>
  <Accordion title="Installation Issues">
    **Problem**: `command not found: envfly` **Solution**: Ensure the
    installation directory is in your PATH: ```bash echo $PATH which envfly ```
  </Accordion>

  <Accordion title="Authentication Issues">
    **Problem**: `Authentication failed` **Solution**: Verify your API key and
    try again: ```bash envfly logout envfly login ```
  </Accordion>

  <Accordion title="Permission Issues">
    **Problem**: `Could not store API key securely` **Solution**: Grant keychain
    access to your terminal application.
  </Accordion>
</AccordionGroup>
